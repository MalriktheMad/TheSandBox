# Assets

This folder is for game art and reference material as Black Candle Lab Zero grows.

- `refs/` keeps screenshots, mockups, and visual references for the current CSS art.
- `sprites/` keeps character, creature, object, and landmark sprites.
- `tiles/` keeps reusable ground, water, cliff, forest, and map tile art.

For now, the game still runs from `index.html`. These folders are here so we can start saving assets without breaking the prototype.
